View - получается все generic views наследуются от этого класса, в этом классе содержатся функции get(), post() и т.п
ListView - Встроенный в Django класс который созадет список чего либо
UpdateView - Встроенный в Django класс в котором происходит редактирования формы, с
использованием которой можно вносить изменения в записи
CreateView - Встроенный в Django класс в котором происходит создание записей на блоге
TempleteView - Встроенный в Django класс, который позволяет использовать шаблоны для нашего приложения
